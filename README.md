# Soporte Mii

Página de soporte e información del proyecto **Portfolio estilo Wii** de Sergio CM.

Sitio estático (HTML/CSS). Se despliega en `soporte.sergiocm.cv`.

## Estructura

- `index.html` — página principal de soporte.
- `soporte-mii.css` — estilos.
- `assets/` — imágenes y recursos.

## Desarrollo local

Abrir `index.html` en el navegador, o servir la carpeta con cualquier servidor estático:

```bash
python -m http.server 8080
```
